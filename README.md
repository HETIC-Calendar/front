# HETIC Calendar - Front-ed

## Commands

### Installation

- `nvm use` : use right node version
- `pnpm i` : install deps

### Developpement

- `pnpm dev` : launch dev server

### Build

- `pnpm build` : build app
- `pnpm preview` : launch preview server

### Others

- `pnpm prepare` : use husky
- `pnpm lint` : linter
- `pnpm format` : format codebase
